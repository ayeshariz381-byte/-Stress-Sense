# 🧠 StressSense — Mental Wellness App

A full-stack Flutter mobile app designed to help university students manage stress and mental health anonymously.

## 🎯 Problem It Solves
Many students suffer from stress, anxiety, and depression silently — afraid to seek help due to social stigma. StressSense provides a private, stigma-free platform to monitor stress and connect with mental health consultants instantly.

## ✨ Key Features

### Student Portal
- 📷 **PPG Heart Rate Sensing** — measures stress using phone camera + flash
- 📊 **Real-time Stress Analysis** — Low / Moderate / High with full history
- 🤖 **AI Counselor Chat** — powered by Gemini AI
- 🫁 **Breathing Therapy** — guided breathing exercises
- 🎵 **Healing Audio** — Quran recitation for calm
- 🚨 **Auto Emergency Alert** — notifies consultant when High stress detected

### Consultant Portal
- 👥 **Live Student Dashboard** — real stress data from Firestore
- 🔴 **Real-time Emergency Alerts** — instant notifications when student is in crisis
- 💬 **Patient Consultation** — AI-simulated patient chat
- 📞 **Direct Call/Message** — instant student contact

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | Flutter (Dart) |
| Backend | Firebase Auth + Firestore |
| AI Chat | Google Gemini API |
| Sensor | PPG via Phone Camera |
| State | Provider |
